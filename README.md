# 지하철 노선도 미션
스프링 과정 실습을 위한 지하철 노선도 애플리케이션

<br>

## 기능 구현 목록

* 경로를 조회하는 API를 구현한다.
* 경로와 함께 총 거리와 요금을 계산한다.
* 환승 노선을 고려한다.

## 도메인

최단경로를 찾는 객체(Path)

* 모든 구간 정보를 알고 있다.
* 최단 경로를 계산한다.

### Usage
#### application 구동
